# 회전기계 진동 신호 기반 이상탐지
이 프로젝트는 회전기계의 진동 데이터에 기반하여 이상 탐지 모델링, 평가 기술을 제공합니다. 회전기계는 산업 분야에서 중요한 장비로 사용되며, 이들의 안정적인 운전을 위해 진동 데이터 모니터링은 필수적입니다. 프로젝트의 코드를 통해 다양한 센서에서 수집된 다변량 시계열 데이터를 활용하여 회전기계의 상태를 실시간으로 평가하고, 이상 징후를 탐지하는데 도움을 줍니다.
